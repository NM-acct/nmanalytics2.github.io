Project: Product Returns

Product Returns Analysis
(Excel + Power Query + Power BI)

This project visualizes product return reasons, quantities, and trends to help optimize return policies and reduce losses.

Objective

To uncover why products are returned, which items are most frequently returned, and how trends change over time. The analysis helps businesses reduce return-related losses.

Tools Used

  • Excel: Data cleaning, conditional formatting, return category tagging
  • Power Query: Automated transformations for messy columns, filters, and date cleanup
  • Power BI: Interactive dashboards with visuals like bar charts, slicers, and KPIs

Sample Insights

  • Top return reason: Defective product
  • Return spikes seen after promotional campaigns
  • High-value items had lower return rates compared to accessories

Outcome

Created a Power BI dashboard summarizing trends, reasons, and return volume over time. Helped simulate real-world client reporting scenarios by preparing and visualizing raw return logs from scratch.
